About me

My name is Marija, a full-stack web developer located in Stuttgart, Germany. I am currently collaborating on a big scale e-commerce web shop that extends for 23 countries and sells 89 000 products, for Cool Services Gmbh in Oxid Esales, PHP, jQuery, JavaScript, PimCore and Docker. Last year I finished a successfull collaboration on a React e-commerce web shop, where I added new components and features and covered the big part of the application with unit and integration tests with Jest.
